CST-323 Project

By Rhese S.

This project was created for CST-323: Cloud Computing in the Spring Semester of 2022 by Rhese, a student at Grand Canyon University. This project was developed under and submitted to Professor Mark Reha.

The point of this project is to have an application that can be used accross multiple cloud provider platforms, in order to learn about different cloud systems. This project was built on Springboot and Thymeleaf in the Java langauge with MySQL supporting it as a backend.
